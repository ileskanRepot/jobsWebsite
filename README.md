# jobsWebsite

## How to get working

You need to make python file named `dbSecter.py` and put this inside it 
```
dbusername = "YOUR DB USERNAME"
dbpassword = "YOUR DB PASSWORD"
dbname = "YOUR DB NAME"
dbhost = "YOUR DB HOST"
dbport = "YOUR DB PORT"
```

Launch with `uvicorn app:app --reload --port 8000`
